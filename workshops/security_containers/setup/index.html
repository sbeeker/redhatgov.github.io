Today’s workshop infrastructure is being run in Amazon EC2. Each student will have a single instance to use. Each of you should have received an email. It has the following information on it. Your student number will be substituted where you see this < student# >. Example: if you are student 21: http://workshopname.21.redhatgov.io

  • Web URL for each of your EC2 instances.

  • Username and password

If you didn’t get one of these emails, let us know so we can give you the information about your lab environment.

Workshop Infrastructure

Each student’s personal EC2 lab is equipped with the following:

  • A directory containing all files used in this workshop.
